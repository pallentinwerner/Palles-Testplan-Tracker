# Run and deploy your app

This contains everything you need to run your app locally.

**Prerequisites:**  Node.js

1. Install dependencies:
   `npm install`
2. Run the app:
   `npm run dev`

or

1. cd /your/front/folder
2. npm run build
3. cp -r build/ /var/www/html
